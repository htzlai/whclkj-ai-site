# 武汉畅连科技有限公司官网

基于 Next.js 构建的企业官网，采用生态绿青蓝配色风格。

## 公司简介

武汉畅连科技有限公司是一家专注于智慧IT解决方案的高新技术企业，致力于为政府、企业提供专业的信息化服务。

## 核心服务

- 🌐 网络建设 - 企业局域网、广域网、无线网络规划与建设
- 🔧 系统集成 - IT基础设施集成、数据中心建设
- ☁️ 云计算服务 - 企业上云、混合云架构设计
- 💻 软件开发 - OA/ERP/CRM等企业管理软件定制开发
- 🛠️ IT运维 - 7×24小时专业运维外包服务

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## 配色方案

- 主色：生态绿 (#059669)
- 强调色：青蓝色 (#0891b2)
- 背景：深邃青绿 (#042f2e)

## 开发

```bash
npm install
npm run dev
```

## 部署

项目通过 Vercel 自动部署。
